About Smart Contact Manager

Smart Contact Manager is a Simple Web-based Application designed to Manage your Persional Contacts in One Place

This Project is builting using Spring Boot, Spring Security, Spring Data JPA, Thymeleaf. It allows users to register, login, and manage their contacts with ease. Each user gets a personal dashboard where they can:

  • Add, View, update, and delete contacts
  • Store email, phone, and description for each contacts
  • Use a secure login system with password encryption

This Application is creating for a learning purpose using Spring Boot
